# short presentation of confion app
=======================

Grundidee & Motivation:
 - Responsive Webdesign
 - Angular Styleguide und Einsatz (von Angular selbst und dem Styleguide) an einem realen Beispiel (das über eine To-Do-App hinausgeht)
 - Backend as a Service mit Datenbank in der Cloud - Firebase als Persistenz-Schicht (SSE - https://en.wikipedia.org/wiki/Server-sent_events)
 - Continuos Integration & Deployment (as a Service) im JavaScript-Umfeld

Demo:
 - Auf Mobile (Simulation im browser)
 - Mit mehreren Clients (Firebase 3 way databinding zeigen)


Technologien & Frameworks:
 - Firebase Datenstruktur zeigen
 - Bootstrap
 - Trello
 - Heroku
 - Travis
 - Firebase
 - Angular
 - Protractor End2End Tests mit Page Object Pattern


Lessons learned:
 - Foundation vs. Bootstrap
 - Firebase Datenstruktur (NoSql, Duplication, No joins) - sehr schnelles Bootstrapping, Produktionseinsatz eher fraglich
 - Autorisierung über Firebase (bietet Service an)
 - Arbeit im Team
 - reales Projektli vs. Tutorial
 - Continuos Deployment sehr hilfreich zum Testen auf Mobilgeräten
 - JSON-Generator kann Testdaten generieren und fungiert sogleich als REST-Endpoint
